# RoadSense AI Documentation

## Overview
RoadSense AI is an end-to-end solution for automated road maintenance.

## Components
1. **Edge Node**: Mobile app for real-time capture.
2. **Vision Engine**: YOLOv8 model for damage detection.
3. **Cloud Core**: Firebase/GCP for data persistence.
4. **Ops Dashboard**: React-based control center for road authorities.
